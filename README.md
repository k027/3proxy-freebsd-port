# 3proxy-freebsd-port
=======================

Overview
--------
FreeBSD port for last version of 3proxy


Installation
------------
git clone https://github.com/k027/3proxy-freebsd-port.git
cd 3proxy-freebsd-port
cp -R 3proxy-k027 /usr/ports/net/

Build
------------
cd /usr/ports/net/3proxy-k027
make
make install
